Utilisation du package Julia MixedModels depuis R

Antoine Jardin, dans un post de 2015, compare les performances du package R lme4 et MixedModels écrit en Julia. La version Julia peut être plus beaucoup rapide.

Douglas Bates, ancien développeur de lme4, et créateur du
package Julia MixedModels montre dans ce post comment appeler ce package depuis R.

Pour apprendre à utiliser MixedModels il y a la documentation et une collection de notebooks très détaillés.

Pour exécuter la suite vous devez installer Julia.
